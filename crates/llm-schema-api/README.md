# llm-schema-api

gRPC and REST API definitions for the LLM Schema Registry.

## Features

- Protocol Buffers definitions
- gRPC services
- REST API endpoints
- Client SDKs generation
- API versioning

## APIs

- **Schema Management**: CRUD operations
- **Validation**: Schema validation endpoints
- **Compatibility**: Compatibility checking
- **Lineage**: Dependency tracking
- **Analytics**: Usage metrics and reports
